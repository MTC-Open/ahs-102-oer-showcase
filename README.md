# AHS 102 OER Course Redesign Showcase

Public GitHub Pages showcase of selected AHS 102 OER course redesign materials and student success highlights.

This version uses cleaned, student-facing excerpts from the course export and a standalone stylesheet that visually echoes the D2L course design. It does not include student records, gradebook content, restricted LMS materials, assessment answers, or private course interactions.

## Files

- `index.html` — showcase homepage
- `styles.css` — standalone public CSS
- `assets/` — banner images extracted from the course package
- `pages/` — selected cleaned course excerpts

## GitHub Pages

Publish from `main` and `/root`.
